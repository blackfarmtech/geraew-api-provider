⚡ Omni Flash — Video Tester

gemini-omni-flash-preview · Interactions API

text_to_video

Gera um vídeo só a partir do prompt.

image_to_video

Anima 1 imagem (primeiro frame) seguindo o prompt.

reference_to_video

Combina 2+ imagens de referência (até 7) num vídeo guiado pelo prompt.

edit

Edita um vídeo existente (enviado inline) seguindo o prompt. A proporção é herdada do vídeo de entrada.

Dica: vídeos curtos/pequenos evitam payloads gigantes. Ideal < 5 MB.

multimodal (custom)

Manda vídeo + imagens + prompt na mesma chamada. Ordem enviada: vídeo → imagens → texto. Escolha a task ou deixe automática.

Sem task: vídeo presente → edit; 2+ imgs → reference; 1 img → image_to_video; só prompt → text_to_video.

Obs.: com task edit a proporção é herdada do vídeo (o campo Aspect é ignorado). Payloads grandes = envio lento; prefira mídia curta/pequena.

Servidor precisa estar rodando: node dist/src/main.js (porta 8012). Auth por header x-api-key.
Obs.: edição conversacional via previous_interaction_id não é suportada por este modelo no path do Vertex (retorna 400) — use o modo edit enviando o vídeo.
